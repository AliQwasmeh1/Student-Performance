# Students Performance EDA Project 

## Objective
Perform Exploratory Data Analysis (EDA) on students performance dataset
and extract meaningful insights.

## Dataset
Source: Kaggle - Students Performance in Exams

## Tasks
- Data Cleaning
- Exploratory Data Analysis
- Visualization
- Insights & Recommendations

## Submission Instructions
1. Fork this repository
2. Create a folder inside `submissions/` with your name
3. Add your Jupyter Notebook
4. Create a Pull Request
